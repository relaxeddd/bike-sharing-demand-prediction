# Bike Sharing Demand Prediction

## Описание проекта
Модель машинного обучения для прогнозирования почасового спроса на аренду велосипедов на основе погодных и календарных условий. Решение позволяет оптимизировать логистику распределения велосипедов, сократить простои и предотвратить дефицит.

## Основные результаты
- **Лучшая модель**: DecisionTreeRegressor с подобранными гиперпараметрами
- **RMSE**: 312 (улучшение на 24% относительно baseline 412)
- **R²**: 0.76 (модель объясняет 76% вариаций спроса)

## Ключевые признаки
| Признак | Важность | Влияние на спрос |
|---------|----------|-------------------|
| Температура | 35% | >20°C → спрос в 5.5 раз выше, чем при <0°C |
| Ночное время | 17% | В 6 раз ниже вечернего пика |
| Влажность | 13% | >80% → падение спроса на 50% |
| Осадки | 2% (в bad_weather) | Дождь → падение на 78% |

## Используемые модели
- DecisionTreeRegressor (лучшая)
- KNeighborsRegressor
- Linear Regression (baseline)
- DummyRegressor
